# tubes-berkom-2

### Cashier system 
Cashier system based on python program that featuring GUI, QR code scanning , and real time inventory analytics using segment tree data structure.

### Feature
<ul>
  <li>Inventory Management</li>
  <li>QR Scanning</li>
  <li>Analytics Dashboard</li>
</ul>

### TechStack
<ul>
  <li>Language: Python 3.x</li>
  <li>
GUI: CustomTkinter </li>
  <li>Image Processing: OpenCV (cv2) & PyZbar (QR)</li>
  <li>Data Handling: Pandas </li>
  <li>Algorithm/DS: Custom Segment Tree </li>
  
</ul>


## Installation
1. Clone or Download this repository.
2. You need Python installed. Run the following command in your terminal to install the required libraries:
    ```sh
    pip install customtkinter pandas opencv-python pyzbar pillow qrcode[pil]
    ```
3. Run the main app
   ```sh
   python main.py
   ```
